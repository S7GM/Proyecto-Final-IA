# Proyecto-Final-IA

### Objetivo: 

El objetivo de este proyecto es desarrollar un chatbot basado en inteligencia artificial que pueda interpretar mensajes en lenguaje natural y ofrecer recomendaciones personalizadas de moda según el evento, clima, formalidad y preferencias del usuario. El proyecto demuestra el uso de modelos de clasificación de texto, manejo de intenciones intents, y flujo conversacional mediante Python.
